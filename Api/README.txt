# Instrucciones para ejecutar el cliente de la aplicación

## Requisitos previos
1. Tener instalado Python 3.x en tu sistema.
2. Tener instalado pip (el gestor de paquetes de Python).
3. Tener instalado Visual Studio Code
4. Tener instalado PgAdmin 4 

## Instalación de dependencias
1. Clona este repositorio en tu máquina local.
2. Abre una terminal (o línea de comandos) y navega hasta el directorio del proyecto.
3. Descarga las dependencias en requirements.
4. Desde la terminal abre vscode
5. Con pgAdmin crea un nuevo proyecto
5. Desde el arachivo env configura la url segun tu proyecto creado anteriormente
6. Abre dos terminales dentro del vscode del proyecto
7. En la primera ejecuta el comando: "bun run dev"
8. En la segunda ejecuta el comando: python cliente.py y podras ejecutar el cliente !!!
